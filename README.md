node-chat
=========

node.jsで作ったチャット



mongoDB
--------
mongoDBのインストールが必要

linux(64bit)の場合
    wget http://fastdl.mongodb.org/linux/mongodb-linux-x86_64-2.4.1.tgz
    tar zxvf mongodb-linux-x86_64-2.4.1.tgz

起動
    mkdir -p /data/db
    cd mongodb-linux-x86_64-2.4.1
    ./bin/mongod

Install
--------
`npm install`

node_chatの起動
--------
`node app.js`
